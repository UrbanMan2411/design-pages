# Прайд Лазер — Landing Page

Лазерная резка неметаллов под ключ.  
Одностраничный лендинг для бизнеса по лазерной резке.

## Технологии

- HTML5 + CSS3 (vanilla, никаких фреймворков)
- CSS Grid, Flexbox, анимации
- Intersection Observer API для scroll-reveal
- Google Fonts: Oswald + JetBrains Mono

## Структура

- `index.html` — полная посадочная страница (HTML + CSS + JS в одном файле)

## Развёртывание

### GitHub Pages

1. Создайте репозиторий
2. Настройте GitHub Pages → Source: `main` branch
3. Сайт доступен по адресу `https://<username>.github.io/<repo>/`

### Локальный сервер

```bash
python3 -m http.server 8081 --directory .
```
